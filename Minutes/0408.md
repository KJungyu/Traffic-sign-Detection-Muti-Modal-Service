(차량 + 도보) -> 교수님이 말한 서비스

메세지 출력 => 휴대폰 (Strimlit or 다른 방법 리엑트) + 음성인식 

----------------------------------------------------------------------------------

Yolo -> 표지판 인식을 통해 정보 추출 -> DSR (음성 인식을 통한 표지판 인식 시작) + 녹화 기능

5. 추가적으로 사진에 대한 정보를 원한다? -> API 호출

-> LLAVA는 텍스트와 이미지를 함께 이해할 수 있는 대규모 다중모달 AI 모델

주요 특징
LLAVA는 이미지와 텍스트를 동시에 처리하여 다음과 같은 작업을 수행

- 이미지에 대한 상세한 설명 생성
- 이미지 내용에 관한 질문에 답변
- 이미지를 기반으로 추론하고 분석
- 이미지와 텍스트 사이의 관계를 이해
